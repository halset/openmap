2. Properties File

The OpenMap Viewer loads properties from an openmap.properties file. It searches for this file in three locations: 1) in the JAR file or class directory, 2) in the installation "share" subdirectory, and 3) in the user's home directory. If it finds the file in any of these places, it loads the properties from it, overriding previous settings. We suggest that you copy the openmap.properties file from the installation "share" subdirectory to your home directory, and edit the settings to your own liking.
