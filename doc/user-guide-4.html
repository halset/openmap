4. Extending the Viewer

A simple way to extend the Viewer is to use different map layers. You can add and remove layers by editing the openmap.properties file.

The source code for the OpenMap Viewer is the com/bbn/openmap/app/OpenMap.java file. It hooks together various MapBean components to create the Viewer application. You can write your own application by copying and changing this file.

Be sure to check the toplevel documentation index for information on the OpenMap core MapBean and Layer APIs.
